Setup Instructions (LXD)
Unix Socket (Local LXD - Recommended):
  1. Select Unix Socket (Local) as Connection Type
  2. Click Save Settings → Test Connection
HTTPS (Remote LXD):
  1. Select HTTPS (Remote) as Connection Type
  2. Click Generate to create client certificates
  3. Copy the Certificate content (auto-filled in the form)
  4. In your LXD terminal, run: lxc config trust add
  5. Paste the certificate when prompted
  6. Enter Server URL: https://<lxd-ip>:8443
  7. Click Save Settings → Test Connection
