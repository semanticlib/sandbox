Setup Instructions (WSL LXD)
  1. Click Generate to create client certificates
  2. Copy the Certificate content (auto-filled in the form)
  3. In your WSL terminal, run: lxc config trust add
  4. Enter a client name (e.g., fastapi-client)
  5. When prompted to paste certificate, paste the certificate
  6. Enter Server URL: https://<wsl-ip>:8443
  7. Click Save Settings → Test Connection

Find WSL IP: Run ip addr show eth0 | grep inet in WSL
